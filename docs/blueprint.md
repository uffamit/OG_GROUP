# **App Name**: AgoraMedAI

## Core Features:

- User Authentication: Secure sign-up and login for both patients and doctors using Firebase Authentication, including email/password and Google OAuth. User roles are strictly managed.
- AI-Powered Voice Assistant: An interactive voice assistant integrated into the patient dashboard that allows users to manage schedules, analyze symptoms, book appointments, and call for emergency services via voice commands. This feature is powered by the OpenAI Realtime API and offers text-to-speech responses. This LLM will use tools to help the user with requests like setting medication reminders.
- Medication Management System: Allows patients to add, edit, and delete medications, set reminders, track dosages, and receive refill alerts. Includes progress tracking with charts and notifications for missed doses.
- Doctor Booking System: Enables patients to search and filter doctors by specialty, location, and availability, view doctor profiles, book appointments, and handle payments. Includes appointment reminders and virtual consultation options.
- Emergency Services Integration: A one-tap emergency button with automatic location detection, direct calls to emergency services (108), and SOS messages to family members, along with quick access to medical history.
- Patient Management: Doctors can view assigned patients, search and filter patients, see patient health summaries, and access medical history, enabling more informed care decisions.
- Compliance Monitoring: Real-time medication adherence tracking with a patient compliance dashboard, visual progress bars, alerts for non-compliance, and automated reminders to patients to enhance adherence.

## Style Guidelines:

- Primary color: Deep teal (#008080) to promote trust and tranquility.
- Background color: Light teal (#E0F8F7), creating a calm and inviting atmosphere.
- Accent color: Blue (#0077b6) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a sans-serif font, known for its clean and modern appearance, suitable for both headlines and body text.
- Use minimalist icons for easy navigation.
- Responsive layout that adapts to different screen sizes, ensuring a seamless experience across devices.
- Subtle animations for a smooth and engaging user experience.
